subscribe!(subscription_mask = :all) {|operation, success, user_data| ...}

Enable event notification.

subscription_mask must be one of :null, :sink, :source, :sink_input, :source_output, :module, :client, :sample_cache, :server, :card, :all or an array containing combination of these items. Default is :all.

unsubscribe!() {|operation, success, user_data| ...}

Disable event notification.

Equal to subscribe!(:null)
