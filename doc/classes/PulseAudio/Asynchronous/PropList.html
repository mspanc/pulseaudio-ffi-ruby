Class PulseAudio::Asynchronous::PropList
In: lib/asynchronous/proplist.rb
lib/asynchronous/errors.rb
Parent: Object

PropList class represents Hash-like object that is internally used by PulseAudio to pass named variables to sinks, modules and other objects.

If you do everything correctly, you should not ever had a need to instantiate it directly.

Valid key names are defined in VALID_KEYS constant. Please look to original C library documentation for key descriptions.

Currently validation is disabled.
